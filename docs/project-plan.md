# ProtectMyPhoto MVP Plan

Official domain: protectmyphoto.in

Public admin email: admin@protectmyphoto.in

## MVP Direction

ProtectMyPhoto should run as a static, browser-based photo and document toolkit for Hostinger shared hosting. User files should remain on-device in the MVP and should not be uploaded or stored.

## Phase 1 Tools

1. Compress Image
2. Resize Image
3. Convert Image
4. Remove Metadata
5. Image to PDF
6. Passport Photo Maker for Indian users

## Phase 1 Pages

1. Home
2. All Tools
3. Tool page for each Phase 1 tool
4. About
5. Contact
6. Privacy Policy
7. Terms & Conditions
8. FAQ
9. Blog landing page

## Later

1. Firebase Auth without blocking file tools.
2. Disposable email blocking for signup.
3. Dashboard for account features.
4. Blog articles for every tool.
5. Premium planning.

## Security Notes

Do not place secrets, API keys with private privileges, SMTP passwords, Firebase Admin SDK credentials, or service account JSON files in frontend code.
