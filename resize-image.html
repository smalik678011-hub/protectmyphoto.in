How to resize an image

  1. Select an image from your device.
  2. Enter width and height in pixels.
  3. Keep aspect ratio on if you want to avoid stretching.
  4. Download the resized image.

Many Indian exam and application forms ask for exact pixel dimensions. Check the official requirement before upload.
