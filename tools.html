Tool FAQ

Use tools without changing your workflow.

Which tool should I start with?

Use Compress Image for file-size limits, Resize Image for exact pixels, and Passport Photo Maker for form-ready ID photos.

Can I use these tools on Hostinger/static hosting?

Yes. The tools run in the browser and do not need a Node.js production server.

Are images saved by ProtectMyPhoto?

No. The MVP tools process files locally in your browser and do not upload or store images.
