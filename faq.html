FAQ

Common questions.

Answers about privacy, files, mobile use, and supported tools.

Are my images uploaded?

No. Image processing happens in your browser. Files are not uploaded to a server for these tools.

Does this work on mobile?

Yes. The site is designed mobile-first, with file size limits to reduce browser crashes on low-memory devices.

Why are some AI tools not included?

Advanced AI tools often need server processing. ProtectMyPhoto starts with private browser tools first.

Can I use passport photos for official forms?

The tool helps prepare dimensions and file size, but you must verify the official requirement before submitting.

Who do I contact?

Email admin@protectmyphoto.in.
